# AGENTS.md

This project is Khmer Typing Adventure, a gamified Khmer typing-learning app.

Priorities:
1. Improve Khmer typing skill, not only game decoration.
2. Accuracy is more important than speed for beginners.
3. CPM is the main speed metric for Khmer typing.
4. Keep the ancient/adventure theme, but readability and learning clarity come first.
5. Keyboard mapping and finger hints must be clear, aligned, and useful.
6. Handle Khmer Unicode carefully. Do not break dependent vowels, coeng/subscript combinations, or combining marks.
7. Store progress cleanly so the app can later support teacher dashboards.
8. Do not remove existing features unless they are broken or replaced with a better version.
9. Run build/lint/tests before final response when available.
